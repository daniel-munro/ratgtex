get /dyneqtl
(dyneqtlGet)
Fetch information for one gene-variant pair in one tissue

Query parameters

geneId (required)
Query Parameter — Ensembl gene ID
tissueSiteDetailId (required)
Query Parameter — Tissue ID
variantId (required)
Query Parameter — Variant ID

Return type

inline_response_200

Example data

Content-Type: application/json
{
  "pValueThreshold" : 0.00220875,
  "tissueSiteDetailId" : "IL",
  "data" : [ -1.11134883833383, -0.314562734207873, 0.556543703618366, 0.522265760640944 ],
  "homoAltCount" : 17,
  "alt" : "A",
  "genotypes" : [ 1, 0, 2, 1 ],
  "pValue" : 0.0000203971,
  "homoRefCount" : 30,
  "geneSymbol" : "Cd300lf",
  "ref" : "G",
  "maf" : 0.41975308641975306,
  "geneId" : "ENSRNOG00000021424",
  "variantId" : "chr10:104362850",
  "hetCount" : 34
}

get /medianGeneExpression
(medianGeneExpressionGet)
