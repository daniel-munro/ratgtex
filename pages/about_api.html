RatGTEx REST API (4.0.0)

Download OpenAPI specification:

Daniel Munro: danbmunro@gmail.com URL: https://danmun.ro

This API provides data for the visualizations in the RatGTEx data portal. This version (v4) provides data for RatGTEx data release v4, which uses the GRCr8 genome assembly.

Median expression for genes in selected tissues

Fetch median expression (TPM) for one or more genes in one or more tissues

query Parameters
geneId
required
Array of strings
Example: geneId=Polr3k,Rtel1,Eno1

One or more gene IDs.

tissueSiteDetailId
Array of strings
Example: tissueSiteDetailId=IL,LHb,NAcc

One or more tissue IDs. If omitted, all tissues are included.

Responses
